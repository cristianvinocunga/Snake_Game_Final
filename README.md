# Snake_Game_Final
Este proyecto es diseñado como una herramienta de integración para la asignatura de Sistemas de Información. El objetivo principal es demostrar la convergencia de cuatro áreas fundamentales de la ingeniería de software desarrolladas a lo largo de 8 semanas:

Arquitectura y Planificación: El software nace de un análisis previo de requerimientos y diseño de diagramas funcionales, asegurando una estructura lógica antes de la codificación.

Estructuras de Datos Dinámicas: Se implementó la gestión de la serpiente mediante Listas Dinámicas, permitiendo el crecimiento del cuerpo en tiempo real basado en la captura de objetos (comida).

Algoritmos de Control y Repetición: El motor del juego utiliza bucles infinitos controlados y estructuras condicionales para la gestión de colisiones y movimiento fluido.

Paradigmas Avanzados y Persistencia: Se integraron técnicas de Programación Funcional para optimizar la lógica de evaluación, y un sistema de Persistencia de Datos que interactúa con el sistema operativo para almacenar récords históricos en archivos locales.

Este proyecto no solo cumple con un fin recreativo, sino que sirve como evidencia del dominio del lenguaje Python y la capacidad de estructurar soluciones informáticas completas, desde la arquitectura macro hasta la implementación técnica final.

OBJETIVO DEL PROGRAMA
Desarrollar una aplicación funcional que aplique conceptos de programación imperativa y funcional, gestionando estados de juego y almacenamiento de datos externos.

FUNCIONALIDADES PRINCIPALES
Movimiento en 4 ejes: Control mediante teclado (W, A, S, D).

Sistema de Crecimiento: Uso de arreglos para gestionar los segmentos de la serpiente.

Persistencia de Récord: Lectura y escritura automática en record.txt para mantener el puntaje más alto.

Detección de Colisiones: Lógica matemática para límites de pantalla y auto-colisión.
